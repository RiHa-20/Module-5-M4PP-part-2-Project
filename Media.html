Semantic Elements
Semantic Elements What they are Where they are used
Semantic elements include the following tags: article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, summary, and time According to W3 Schools, semantic elements are elements with a meaning. Semantic elements are used to define different parts of a webpage, they clearly define the content unlike non-semantic elements.
What else is new to HTML 5?
What is New to HTML 5? There is a lot that is new to HTML 5 coding protocol. There has been a lot of new additions including the introduction of the audio and video tags that allows us to embed a video or audio onto our websites and CSS can be used on these tags to change a variety of different properties to the video or audio. Additionally, the header and footer tags where added in HTML 5 and is no longer needed to be identified with a div tag allong with a class. The addition of the already defined semantic tags was also introduced in HTML 5. Additionally, HTML 5 introduced the progress element, placeholder attribute and the email attribute.
